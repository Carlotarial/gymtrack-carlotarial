# GymTrack - TFC 2026

GymTrack es una aplicación móvil de fitness desarrollada con React Native y Expo, enfocada en la simplicidad visual y la gestión de múltiples perfiles de usuario de forma local.

## Características Principales
* **Gestión Multicuenta:** Creación y cambio de perfiles en un mismo dispositivo.
* **Persistencia Local:** Almacenamiento de datos mediante AsyncStorage (sin base de datos externa).
* **Tematización Dinámica:** Modo claro y oscuro persistente por cada perfil de usuario.
* **Seguimiento de Hábitos:** Registro de hidratación, rachas de entrenamiento y métricas físicas.

## Stack Tecnológico
* **Framework:** React Native (Expo SDK)
* **Lenguaje:** TypeScript
* **Navegación:** Expo Router
* **Animaciones:** React Native Reanimated

## Instrucciones de Uso

1. Asegurarse de tener instalado **Node.js**.
2. Abrir una terminal en la carpeta raíz del proyecto.
3. Instalar las dependencias necesarias:
   `npm install`
4. Iniciar la aplicación:
   `npx expo start`

## Estructura del Proyecto
* **/app**: Rutas y pantallas de la aplicación.
* **/context**: Lógica de usuario y sistema de temas.
* **/assets**: Imágenes, logos e iconos.
* **/components**: Componentes visuales reutilizables.
